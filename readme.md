## Dev Dependencies
  - jade
  - scss
  - jasmine
  - ruby(2.3.0)


## External Libraries
  - AngularJS(1.5.7)
  - Fontawesome(4.7.0)


## Package App
```bash
npm run package
```
Compiled files are place in dist/

## Run App
```bash
npm run start
```
or, open file dist/index.html in browser.

## Run Tests Once
```bash
npm run test-once
```

## Watch Js files and Run Tests
```bash
npm run test
```


## Watch for changes and Recompile
```bash
npm run watch
```
This will re-compile app on any change to scss, js or jade files.

Compiled files are place in dist/
